▸ loadVolumeAtlasData(imageArray, callback)

load per-channel volume data from a batch of image files containing the volume slices tiled across the images

loadVolumeAtlasData(imageArray: Array<{name: string, channels: Array<number>}>, callback: PerChannelCallback): Object<string, Image>
Parameters
imageArray (Array<{name: string, channels: Array<number>}>)
callback (PerChannelCallback) Per-channel callback. Called when each channel's atlased volume data is loaded
Returns
Object<string, Image>: a map(imageurl : Image object) that should be used to cancel the download requests, for example if you need to destroy the image before all data has arrived. as requests arrive, the callback will be called per image, not per channel
Example
loadVolumeAtlasData([{
    "name": "AICS-10_5_5.ome.tif_atlas_0.png",
    "channels": [0, 1, 2]
}, {
    "name": "AICS-10_5_5.ome.tif_atlas_1.png",
    "channels": [3, 4, 5]
}, {
    "name": "AICS-10_5_5.ome.tif_atlas_2.png",
    "channels": [6, 7, 8]
}], mycallback);

maxval (number) 0..1, should be greater than minval
isOrthoAxis (boolean) is this an orthographic projection or just a clipping of the range for perspective view
